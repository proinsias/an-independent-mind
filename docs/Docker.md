# Docker

## diff

Inspects changes to files or directories on a container's filesystemÂ since the container was created. Three different types of change are tracked:

-   A file or directory was added (A).
-   A file or directory was deleted (D).
-   A file or directory was changed (C).

```shell
docker diff <container>
```

## stats

Displays a live stream of container(s) resource usage statistics.

```shell
docker stats
```
