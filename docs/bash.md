# bash

## Shortcuts

```shell
!!  # Repeat last command.
!$  # Repeat the last argument of the last command.
!:1-$  # Repeat all the arguments of the last command.
!*  # Equivalent, but without the possibility of subselection of arguments.
!$:h  # The :h applies `dirname` to the argument.
```
